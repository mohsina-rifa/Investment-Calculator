# **Investment Calculator**

A professional web-based utility designed to calculate and project long-term investment growth. This project focuses on component-driven architecture, state management, and real-time data derivation.

## **🚀 Features**

* **Interactive Inputs:** Fetch and store user-provided investment parameters, including initial and annual contributions.  
* **Dynamic Calculations:** Derive yearly results using a specialized utility function to track portfolio performance over time.  
* **Structured Data Output:** Display financial projections in a clean, accessible HTML table utilizing standard `<thead>` and `<tbody>` elements.  
* **Validation Logic:** Conditional rendering that displays a feedback message if an invalid duration (e.g., less than 1 year) is entered.

## **🛠️ Technologies Used**

* **Frontend:** React / HTML5 & CSS3  
* **Logic:** JavaScript (ES6+) for state handling and complex data calculations.  
* **Styling:** Semantic HTML and custom CSS for clear, responsive financial data presentation
